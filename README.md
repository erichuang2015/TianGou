# tiangou

一个舔狗的终极日记，采用json文件存储，没有数据库。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### 后台文件

在public的admin.php

### 部署

直接打包，把dist文件上传服务器，给admin.php文件赋予读取写入权限

linux权限如下：

```
chmod 777 admin.php
```
